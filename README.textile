h2. INSTALLATION

@gem install gemhub@

h2. USAGE

Create your gem's project structure with:

@gemhub <gem-name>@
For more options, execute: @gemhub -h@

Then edit Rakefile:

<pre>
GEM_VERSION = "0.1.0"
SUMMARY = "FIXME Summary"
DESCRIPTION = "FIXME Description"
AUTHOR = "FIXME Author"
EMAIL = "FIXME Email"
HOMEPAGE = "FIXME HomePage"
[...]
</pre>

And finally execute:
@rake make_spec@

To create the gemspec file.

In project folder, use:
@rake -T@ to see all available tasks.
